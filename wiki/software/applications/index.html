Applications Driver station The term Driver station can be used to describe the laptop's we use to operate the robot or the software that runs on them. The laptop's are used only for operating the robot and should not be used for anything else. The driver station software is the software that runs on the driver station laptops and communicates and controls the state of the robot such as setting operating mode and enabling/disabling the robot. more info on what the driver station software can show and do can be found here: https://docs.wpilib.org/en/stable/docs/software/driverstation/driver-station.html CTRE Phoenix Tuner CTRE Phoenix tuner is a program that allows us to see the status of all CTRE devices on the CAN bus, this includes any CTRE motor controllers (TalonSRX or TalonFX) and other components like the PDP and PCM. It allows you to view set the ID's used to address devices on the CAN bus which are required to use those devices in code and is also a useful tool in finding wiring issues in the CAN bus. To use Phoenix tuner you just need to be connected to the robots network. REV Hardware Client REV hardware client is a program that allows us to see the status of all REV devices on the CAN bus, for us this is just the sparkMax motor controller. It also you to view and set the ID's used to address devices on the CAN bus which are required to use those devices in code. To use the REV Hardware client you need to connect to a REV device on the can bus with a usb-c cable.
